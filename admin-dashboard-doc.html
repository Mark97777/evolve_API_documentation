Evolve Admin Dasboard Documentation

This API will return the necessary data needed on Evolve Admin dashboard.

Please note!

  1. Evolve Admin, will register like any normal partner registration (but will receive a unique priviledge afterwards).
  2. As a top level admin, sees all data of all partners.
  3. There are only two endpoints available for the Admin dashboard.

Main Access point

All requests must point to this url.

https://xowiepos.com/app/zver1/public/

Authentication

All requests must include a valid API key in the API-KEY header.

API-KEY:
