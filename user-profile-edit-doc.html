User Profile Edit Documentation

This documentation helps you to edit and get the user's detail credentials for Evolve Partners.

Points to note

  1. Requires that user is already signed in
  2. Profile photo is optional during updating.
  3. Input field for profile photo should have key / name value of user_photo.(very important for image upload to work.)

Main Access point

All requests must point to this url.

https://xowiepos.com/app/zver1/public/

Authentication

All requests must include a valid API key in the API-KEY header.

API-KEY:
POST

?action=update-user

This endpoint will update the user profile information.
NOTE: Updating any user information is optional.

Request Data

Name Type
user_id string / int
first_name string
last_name string
telephone string
company string
bio string
country string
city string
user_photo file

JSON Structure


Form must inlcude all required data above
