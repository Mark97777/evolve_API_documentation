API Documentation

NOTE: The SMS call may delay due to network issues or other factors. Also, please limit number of calls to the SMS endpoint (It has limited credit for testing).

POST

?action=generate-otp

Generate a One-Time Password (OTP).

Request Data

Name email Description
recipient string Email address of the recipient.

JSON Structure


// Email data to be sent (PHP sample)
$data = [
    'recipient' => "user_email@gmail.com"
];

Response


		SUCCESS RESPONSE
{
  "status": "success",
  "otp": "123456"
}
		ERROR RESPONSE
{
  "status": "error",
  "message": "Failed to generate OTP."
}
POST

?action=verify-otp

This endpoint verifies the generated OTP. The endpoint requires the email and generated OTP so it can verify it. Generated OTP expires after 5 minutes.

Request Data

Name Type Description
email string The recipient's email used to generate the OTP previously.
otp string The generated OTP to verify.

JSON Structure


// Request data
$data = [
    'email' => "user_email@gmail.com", // the email used to request the OTP
    'otp'      => "generated_OTP_here" // The generated OTP associated with the email
];

Response


		SUCCESS  RESPOSNE
{
    "status": "success",
    "message": "OTP verified successfully."
}
		ERROR  RESPOSNE
{
    "status": "error",
    "message": "OTP has expired."
}
POST

?action=email-otp

Send an email to a recipient.

Request Data

Name Type Description
recipient string Email address of the recipient.
subject string Subject of the email.
body string Body content of the email (Must contain the OTP).

JSON Structure


// Email data to be sent (PHP sample)
$data = [
    'recipient' => "user_email@gmail.com",
    'subject'   => 'EVOLVE OTP',
    'body'      => "Your OTP to send is 837462."
];

Sample Nodejs POST


const axios = require('axios');

// API URL
const apiUrl = 'http://localhost/consultcaresolutions/app/portal/email-api/public/?action=email-otp';

// Email data to be sent
const data = {
    recipient: 'your_email@gmail.com',
    subject: 'EVOLVE OTP',
    body: 'Your OTP is 837462.' // Insert the generated OTP in the body.
};

// Headers including the API key
const headers = {
    'Content-Type': 'application/json',
    'API-KEY': 'YOUR_API_KEY'
};

// Make a POST request using axios
axios.post(apiUrl, data, { headers })
    .then(response => {
        // Check if the response is valid JSON
        if (response.data) {
            console.log(JSON.stringify(response.data, null, 2)); // Pretty-print the JSON response
        } else {
            console.error('Invalid JSON response from API');
        }
    })
    .catch(error => {
        // Handle errors
        if (error.response) {
            // The request was made and the server responded with a status code
            console.error('Server responded with an error:', error.response.data);
        } else if (error.request) {
            // The request was made but no response was received
            console.error('No response received from the server:', error.request);
        } else {
            // Something happened in setting up the request
            console.error('Error setting up the request:', error.message);
        }
    });

Response


		SUCCESS  RESPONSE
{
    "status": "success",
    "message": "Email sent successfully."
}
		ERROR  RESPONSE
{
    "status": "error",
    "message": "Email not sent."
}
